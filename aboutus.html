Meet the Developer

Hello! I'm a grad student at USF, pursuing a master's degree in AI & Business Analytics, and I'm also a developer with a passion for pop music and glitter.

Developing this website was a rewarding project focused on creating a vibrant, user-friendly platform inspired by modern fashion trends, using a pink, gold, and white color scheme.

I started with a HTML5 structure to ensure easy navigation. Then, I sprinkled some CSS magic to add stylish gradients and responsive design so it looks great on any device. To make things interactive, I used JavaScript for features like a handy hamburger menu, form validation, and CRUD operations for managing wardrobe items, keeping everything saved in local storage. Plus, I included Chart.js on the Analytics page to visually showcase user trends.

Check out my source code on GitHub.
